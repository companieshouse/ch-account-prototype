Scenario One

A director sets up a CH account and has their authentication code. They add their company to their user account and then later add an authorised person to the company.

Scenario Two

A 'director/PSC/Authorised person' has no authentication code. The company has not already been linked to a user account. The user will need to request the authentication code before they can add the company to the account.
