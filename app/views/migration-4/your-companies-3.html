Digital authorisation

For security reasons, as WebFiling now uses GOV.UK One Login to sign in, you'll need to enter each company authentication code to file online. You'll have the option to store the code, so you do not have to enter it again (for most online filings).

If you do not have the authentication code, you'll either need to ask someone who is digitally authorised to add you, or request the authentication code (opens in a new tab).
