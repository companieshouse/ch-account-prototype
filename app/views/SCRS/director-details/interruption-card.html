Digital authorisation

You can now digitally authorise directors to file information online when you incorporate a company, instead of receiving a code in the post.

This means directors can:

  • instantly file online as soon as the company is incorporated
  • file most information online without an authentication code
  • digitally authorise other people to file online for the company

Digital authorisation is quicker, paperless and more secure.

{{ govukButton({ text: "Authorise directors digitally" }) }}

If you do not want to digitally authorise directors, you can choose to receive an authentication code the post instead. It will be sent to the registered office address once the company is incorporated. It takes up to 5 working days to arrive.
