You're about to set up online filing authorisation for the company

You can digitally authorise directors to file information online with Companies House when you incorporate a company, instead of receiving a code in the post.

This means directors can:

  • instantly file online as soon as the company is incorporated
  • file most information online without an authentication code
  • digitally authorise other people to file online for the company

Digital authorisation is quicker, paperless and more secure.

It's more secure because:

  • we'll send a security code to your mobile phone or email address each time you sign in to file online for the company
  • you'll know exactly who is digitally authorised to file online
