Sign in to WebFiling: lenders only

You will need:

  • the company number
  • your lender authentication code
  • your presenter authentication code

If you're not a lender, you need to sign in using GOV.UK One Login instead.

What you can do in WebFiling

In WebFiling, you can:

  • file your company’s annual accounts and confirmation statement
  • file director and secretary information
  • sign up to get email reminders when your accounts and confirmation statement are due
  • digitally authorise people to file for your company online

Help with company number
This is 8 characters long. You can find it by searching for the company on the Companies House register.
Help with authentication codes

A lender authentication code is an 8 character code. It's required if you need to file charge (mortgage) documents online for other companies.

A presenter authentication code is an 11 character code. It's required when using software to make a filing.

If you're a lender and do not have the codes, you'll need to apply to register as a lender to file charge documents for a company.

If you've previously registered as a lender but no longer have the codes, contact us at enquiries@companieshouse.gov.uk
